# 上下文环境和作用域

## 执行上下文
> 给执行上下文环境下一个通俗的定义——在执行代码之前，把将要用到的所有的变量都事先拿出来，有的直接赋值了，有的先用 undefined 占个空。

在一段 js 代码拿过来真正一句一句运行之前，浏览器已经做了一些 **「准备工作」**，其中就包括对「变量的声明」，而不是赋值。「变量赋值」是在赋值语句执行的时候进行的。可用下图模拟：
